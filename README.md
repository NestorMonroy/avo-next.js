### 🤖 Guía Rápida

1.  **Empieza a desarrollar.**

    Instala dependencias

    ```sh
    npm install o yarn
    ```

    Inicia el proyecto

    ```sh
    npm run dev o yarn dev
    ```

    El sitio estará disponible en http://localhost:3000.
